Setting Up

  • import node.js if you don't already have it
  • check which versions of node.js and npm that you have (npm -v, node -v)
  • import the Angular CLI npm install -g @angular/cli
  • test to see if it worked using ng -v
  • if it didn't work, which mine weirdly didn't, try this: npm uninstall -g @angular/cli then npm cache verify then ng -v and if that still doesn't work do this: alias ng="/Users/username/.npm-global/bin/ng" ...shoutout stack overflow lol
  • start your new project using ng new projectname
  • cd into it (you DEF know that but I legit forget to everytime lol it's fine i'm fine)
  • npm install !!! then ng build
  • npm install again for shits n giggles
  • ng serve -o to run it on your local host
  • generate components using ng generate component componentName

Tips that save my life:
